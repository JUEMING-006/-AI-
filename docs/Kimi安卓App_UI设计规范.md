# Kimi 安卓 App 完整 UI 设计规范

> 本文档为 Kimi 安卓手机 App 的完整 UI 布局与设计规范，涵盖首页、侧边栏、对话区、输入框、Logo悬浮球、动画等所有界面元素。
> 目标：让开发者可以直接参照本文档，完整实现一个类 Kimi 风格的安卓 AI 助手应用。

---

## 一、整体架构

### 1.1 屏幕结构

设计稿基准尺寸：1080×1920px（标准安卓手机）

```
┌─────────────────────────────────────────┐
│ ① 顶部导航栏 (56dp)                     │
│   [☰侧边栏] [模型选择器▼] [发现] [头像] │
├─────────────────────────────────────────┤
│ ② Taskbar 工具栏 (44dp)                 │
│   [通用] [文档] [网站] [PPT] [表格]     │
├─────────────────────────────────────────┤
│                                         │
│ ③ 对话内容区（无边界多模态容器风格）      │
│   - 用户消息（右侧，简洁卡片）           │
│   - AI回复（左侧，宽幅容器，Markdown渲染）│
│   - 思考过程（可折叠）                   │
│   - 工具调用分组（可折叠容器）            │
│   - 追问建议卡片组                       │
│                                         │
├─────────────────────────────────────────┤
│ ④ 底部输入区 (56-120dp)                 │
│   [+附件] [输入框...] [🎤语音] [▶发送]  │
└─────────────────────────────────────────┘
```

### 1.3 导航模式

- 首页可**右滑**打开侧边栏
- 侧边栏也可通过左上角汉堡按钮打开
- 对话页面通过返回按钮回到首页
